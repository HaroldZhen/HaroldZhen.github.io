心得

每次git commit都會產生一個Patch，

git rebase 整理Patch會改變Hush值

git reset https://git-scm.com/book/en/v2/Git-Tools-Reset-Demystified.

  • 移動HEAD分支的指向 git reset --soft

    只是移動HEAD

  • 使索引看起來像HEAD git reset --mixed

    移動HEAD，還原Stage區

  • 使工作目錄看起來像索引 git reset --hard

    移動HEAD，….還原工作目錄
